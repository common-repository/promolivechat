== Installation ==

1. Upload `promolivechat.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Options page (Settings -> PromoLiveChat) and setup your support operators Jabber/XMPP accounts.
